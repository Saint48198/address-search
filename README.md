# Address Search

A simple Node.js application using Express to provide address search functionality.

## Features

- REST API for address searching
- Uses `express` for server
- Fetches data using `node-fetch`

## Requirements

- Node.js (v16+ recommended)
- npm

## Installation

```bash
npm install
